# Enterprise-grade robots.txt for a Next.js App Router site.
#
# What we allow:
# - Everything under /_next/ (chunks, fonts, optimized images, RSC payloads).
#   Blocking these breaks Google's rendering crawler, which needs to fetch
#   JS/CSS/image resources to render pages for indexing. The earlier
#   `Disallow: /_next/` directive caused "Blocked by robots.txt" reports in
#   Google Search Console because `/_next/image/` (optimized-image endpoint)
#   and `/_next/data/` (RSC hydration payloads) were inaccessible.
# - Static assets in /static/ and /public/ (served at root).
#
# What we block:
# - /api/ — transactional endpoints with no indexable content.

User-agent: *
Allow: /
Disallow: /api/
Disallow: /design-preview
