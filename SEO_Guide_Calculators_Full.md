# SEO Keyword Placement & FAQ Guide for Astrology/Numerology/Vastu/Muhurat Tools

**Purpose:** SEO optimization for **minimal-content calculator pages** (Title, Meta, H1, Subtitle, FAQ, Schema).

---

## GENERAL PAGE STRUCTURE RULES

1.  **Title Tag:** `[Primary Keyword] | [Brand Name]` (60 chars max)
2.  **Meta Description:** `Calculate [Primary Keyword]... [Benefit]. Free online.` (160 chars max)
3.  **H1:** Exact `[Primary Keyword]`
4.  **Subtitle:** 1-2 sentences explaining the tool + 1 secondary keyword.
5.  **FAQ Section:** 4-6 Questions using long-tail keywords.
6.  **Schema:** `SoftwareApplication` + `FAQPage` JSON-LD.

## 5. SCHEMA MARKUP TEMPLATES

### A. SoftwareApplication Schema (For all calculator pages)
*Place in `<head>`*
```json
{
  "@context": "https://schema.org",
  "@type": "SoftwareApplication",
  "name": "[Insert H1 Here]",
  "description": "[Insert Meta Description Here]",
  "applicationCategory": "UtilityApplication",
  "operatingSystem": "Web",
  "offers": {
    "@type": "Offer",
    "price": "0",
    "priceCurrency": "USD"
  }
}
```

### B. FAQPage Schema (For all calculator pages)
*Place in `<head>`*
```json
{
  "@context": "https://schema.org",
  "@type": "FAQPage",
  "mainEntity": [
    {
      "@type": "Question",
      "name": "[Insert Question 1]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Answer goes here..."
      }
    },
    {
      "@type": "Question",
      "name": "[Insert Question 2]",
      "acceptedAnswer": {
        "@type": "Answer",
        "text": "Answer goes here..."
      }
    }
    // Repeat for all 6 questions
  ]
}
```
